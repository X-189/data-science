**Calorie Intake Tracker using Excel**

📌 Project Overview
 
* This project focuses on tracking daily calorie intake using Excel
* Helps users record food consumption and monitor diet
* Calculates total calories and compares with  a target value
* Provides visual insights using charts and dashboards

🎯 Objectives

 * To track daily food consumption 
 * To calculate total calorie intake 
 * To compare intake with target calories (e.g., 2000 kcal/day) 
 * To identify eating patterns and trends 
 * To visualize data using charts and dashboards 

⚙️ **Tools & Technologies**
Microsoft Excel
Pivot Tables
Charts (Bar, Pie, Line)
**Excel Formulas:**
   * SUMIF (calculate total calories)
   * IF (comparison with target)
   * Maximum (weekly analysis)


🔄 Workflow

**Input → Data Storage → Processing → Analysis → Visualization**

Step 1: Data Entry (Date, Food Item, Category, Calories)
Step 2: Data Processing using formulas
Step 3: Calculation of daily total calories
Step 4: Comparison with target calories
Step 5: Analysis using PivotTables
Step 6: Visualization using charts


📈 Features
* Daily calorie tracking 
* Target vs actual comparison 
* Weekly analysis 
* Category-wise calorie breakdown 
* Visual dashboards
